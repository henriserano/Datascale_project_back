2. Processus de recrutement d'un nouveau collaborateur
Le département RH publie une offre d'emploi et reçoit les candidatures via un portail dédié. Une présélection est réalisée pour identifier les profils pertinents. Les candidats retenus passent un entretien téléphonique, suivi d’un entretien physique ou vidéo avec les managers. Une fois le choix effectué, une proposition d’embauche est envoyée. Après acceptation, le candidat est intégré dans le processus d'onboarding.
